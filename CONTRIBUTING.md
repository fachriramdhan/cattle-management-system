## Cara Berkontribusi

1. Fork repository ini
2. Clone fork Anda
3. Buat branch baru untuk fitur/bugfix Anda
4. Commit perubahan Anda
5. Push ke branch Anda
6. Buat Pull Request

## Code Style

- Ikuti C# Coding Conventions
- Gunakan PascalCase untuk class names
- Gunakan camelCase untuk variable names
- Tambahkan XML comments untuk public methods

## Pull Request Process

1. Update README.md jika diperlukan
2. Update dokumentasi jika menambah fitur baru
3. Pull request akan di-review oleh maintainer
4. Setelah approved, akan di-merge ke main branch
